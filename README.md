# meeTeam: 실력 기반 사이드프로젝트 매칭과 자동 포트폴리오 플랫폼

## 1. 서비스 개요

meeTeam은 **사이드 프로젝트를 하고 싶은 사람들을 위한 팀 매칭 플랫폼**이다. 단순한 “구인·구직식 매칭”이 아니라, **실제 작업 단위인 '마이크로 태스크(Micro-task)'를 중심으로 실력과 신뢰도를 평가하고, 이를 통해 가장 잘 맞는 팀원을 자동 추천하는 AI 기반 서비스이다.**

또한 참여자의 활동 기록과 산출물을 자동으로 수집해 **AI가 실시간 ‘Auto-folio(자동 포트폴리오)’를 생성**해 준다. 사용자는 팀 프로젝트를 경험하는 즉시 개인 브랜딩 자산이 쌓이게 된다.

meeTeam은 사이드 프로젝트에서 흔히 발생하는 문제(주도자 소실, 업무 비대칭, 레벨 mismatch, 실력 불투명, 산출물 관리 부재)를 해결하고, **프로젝트 경험이 경력으로 전환되지 못하는 문제를 구조적으로 해결**하는 것을 목표로 한다.

---

## 2. 핵심 기능

### 2.1 Task-Master AI: 프로젝트 자동 분해 및 우선순위 생성

사용자가 프로젝트 아이디어 또는 목표를 입력하면, Task-Master AI가 이를 분석하여 다음을 자동 생성한다.

- 전체 프로젝트를 **마이크로 태스크 단위로 자동 분해**
- 중요도·선행 관계·난이도 기반 **우선순위 매트릭스 자동 생성**
- 역할(기획·디자인·프론트·백엔드 등)별 추천 태스크 제공
- 태스크별 예상 소요 시간 자동 추정

이를 통해 **초기 설계 부담을 줄이고, 초보자도 구조화된 프로젝트 진행이 가능하도록 한다.**

---

### 2.2 신뢰 기반 팀 매칭: Collaboration Profile

meeTeam은 기존의 포트폴리오·자기소개 중심 매칭 방식 대신, **실제 프로젝트 실행 중 기록되는 데이터로 팀원을 평가**한다.

#### 평가 요소:

- 태스크 수행 속도 및 마감 준수율
- 협업 태도(응답 속도, 커뮤니케이션 빈도 등)
- 동료 평가지표(마일스톤 종료 시 익명 peer review)
- 태스크 난이도 대비 성취도

이 기록들은 ‘Collaboration Profile’로 축적되어, AI가:

- 가장 잘 맞는 스타일의 팀원 추천
- 조직에서 말하는 *“일 잘하는 사람”*의 정량화

를 가능하게 한다. **서류로는 보이지 않는 협업 역량을 데이터로 드러내는 것이 핵심 가치이다.**

---

### 2.3 Auto-folio: 자동 포트폴리오 생성기

유저가 태스크를 수행하면 meeTeam은 다음 정보를 자동 수집·정리한다.

- 수행한 태스크 로그(텍스트/코드/이미지/작업 산출물)
- 문제 해결 과정 Q&A 기록
- 사용 기술 스택
- 팀 프로젝트의 전체 맥락과 유저의 기여 구간

이 데이터를 기반으로 AI가 자동으로:

- **프로젝트 포트폴리오 페이지**
- **작업 요약 문서**
- **성과·지표 기반 결과 보고서**
- **면접 대비 스토리라인**

까지 생성해 준다.

즉, **“프로젝트만 하면 포트폴리오가 자동으로 만들어지는”** 구조이다.

---

## 3. 해결하는 문제들

### 3.1 팀 매칭 실패

- 실력 보이기 어려움
- 자기소개 허풍/과소평가 문제
- 역할 불명확

→ meeTeam은 **실제 협업 데이터를 기반으로 팀원을 추천**하여 실패 확률을 크게 줄인다.

### 3.2 프로젝트 중도 하차 문제

- 리더 의존형 진행
- 일정 지연 및 책임 분배 모호

→ **마이크로 태스크와 AI PM 기능으로 프로젝트가 자동 진행되는 구조**를 만든다.

### 3.3 포트폴리오 작성 부담

- 내용 정리 귀찮음
- 증빙자료 누락
- 객관적 지표 부족

→ Auto-folio가 **실시간 자동 포트폴리오 생성**.

---

## 4. 타깃 사용자

- 사이드 프로젝트를 시작하고 싶은 대학생·취준생
- 실력있는 동료를 찾는 독립 개발자/디자이너
- 경력 전환을 위해 포트폴리오가 필요한 사람
- 팀 단위 인턴십 프로그램 운영 기관
- 생산성·협업 지표 기반의 HR 관리를 하고 싶은 스타트업

## 6. 비전

meeTeam은 “사이드 프로젝트 문화”를:

- 더 쉽게 시작할 수 있고
- 더 투명하고 공정하며
- 노력한 만큼 기록이 남고
- 경력으로 자연스럽게 연결되는 구조

로 재편하는 것을 목표로 한다.

장기적으로는 **개인의 "협업 데이터"가 경력의 핵심 지표가 되는 생태계**를 구축하며, 이를 대학·기업·교육기관까지 확장하는 것을 비전으로 한다.

---

## 7. 결론

meeTeam은 단순 매칭 사이트가 아니라,  
**“협업 역량을 중심으로 한 새로운 경력 빌딩 플랫폼”**이다.  
더 이상 프로젝트 경험이 사라지지 않고,  
모든 기여가 기록되고,  
기록이 실력이 되고,  
실력이 기회를 만드는 구조를 만드는 것이 궁극적인 목표이다.
